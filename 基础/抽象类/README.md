--抽象类----------------------------------------------------------
------------------------------------------------------------------
--注意------------------------------------------------------------
抽象类,抽象方法必须使用abstract修饰符来修饰
抽象方法不能有方法体
抽象类不能被实例化
抽象类的构造器不能用于创建实例,主要是用于被其子类调用
抽象类可以不包含抽象方法
抽象类可以包含成员变量,方法(普通方法和抽象方法都可以),构造器,初始化块,内部类(接口,枚举)
含有抽象方法的类只能被定义成抽象类
